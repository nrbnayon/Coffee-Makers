# ByteBlaze

ByteBlaze is a web application for managing and bookmarking blogs. It allows users to save their favorite blogs, view them later, and delete them if needed.

## Features

- Read Blogs
- View a list of bookmarked blogs
- Delete individual bookmarks or all bookmarks at once
- Responsive design for mobile and desktop

## Technologies Used

- React.js
- React Router DOM
- LocalStorage API for data persistence
- Tailwind CSS for styling
- React Icons for delete button icon
